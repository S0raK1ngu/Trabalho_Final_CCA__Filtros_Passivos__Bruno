# Trabalho_Final_CCA__Filtros_Passivos__Bruno
Criação de uma ferramenta em Python para projetar um crossover passivo de duas vias , utilizando filtros Butterworth de 2ª ordem. O software calcula os valores ideais de $L$ e $C$, seleciona componentes comerciais e gera gráficos de Bode comparativos para analisar o desempenho real vs. ideal.
